# Ucode_EndGame
## Usage:
to install game: 
```bash
> make
```
to reinstall game: 
```bash
> make reinstall
```
to uninstall game:
```bash
> make uninstall
```
to uninstall binary file: 
```bash
> make clean
```
to execute game: 
```bash
> ./endgame
```
